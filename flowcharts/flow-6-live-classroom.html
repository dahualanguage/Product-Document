6. Live Classroom

Content Teacher broadcasts live audio → Real-time transcription → ESL Students receive clickable transcript → Word-level language support → Difficulty events feed Data Dashboard.

A Live Session & Real-Time Transcription Product Spec

100%
flowchart TD
    START([Content Teacher opens Live Session]):::start --> ROLE{User role?}:::decision

    ROLE -->|CONTENT_TEACHER /
ESL_SUBJECT_TEACHER| AUTH["Authorized to broadcast
Live session controls visible"]
    ROLE -->|ESL_TEACHER| VIEW["Can view student transcript activity
but does not broadcast"]
    ROLE -->|ESL_STUDENT| STUDENT_JOIN["Joins live session
as transcript listener"]
    ROLE -->|Other roles| NO_ACCESS([No live session access]):::endnode

    AUTH --> MIC["Enable Microphone
Browser permission required"]
    MIC --> MIC_CHECK{Microphone granted?}:::decision
    MIC_CHECK -->|No| MIC_FAIL["Browser blocked microphone
Must allow in browser settings"]:::warn
    MIC_CHECK -->|Yes| STREAM["Audio stream active
Sent to AssemblyAI via WebSocket"]

    STREAM --> ASM["AssemblyAI processes audio
Real-time speech-to-text"]:::ai
    ASM --> ABLY["Transcription pushed to
Ably pub/sub channel
Channel: session-specific"]
    ABLY --> STUDENTS["All subscribed students
receive live transcript"]

    STUDENT_JOIN --> SUB["Subscribe to Ably channel
for current session"]
    SUB --> STUDENTS

    STUDENTS --> DISPLAY["ClickableTranscript component
renders transcript in real time"]
    DISPLAY --> SESSION_END{Teacher ends session?}:::decision
    SESSION_END -->|No| STREAM
    SESSION_END -->|Yes| SAVE["Session transcript saved
Available for review"]
    SAVE --> DONE([Live session complete]):::endnode

    VIEW --> MONITOR["Monitor student difficulty events
See which words students click"]
    MONITOR --> DONE

B ClickableTranscript & Word-Level Support Product Spec

100%
flowchart TD
    START([Student sees live transcript]):::start
    START --> RENDER["ClickableTranscript component
Every word is a tappable element
Transcript scrolls as teacher speaks"]

    RENDER --> TAP["Student taps an unknown word"]
    TAP --> LOOKUP["lookupWord GraphQL query
Sends word + surrounding context"]:::ai

    LOOKUP --> POPUP["Support popup appears:
Translation · Definition
Pinyin · Example sentences"]
    LOOKUP --> EVENT["Difficulty event recorded:
word · timestamp · studentId
sessionId · contextSentence"]

    POPUP --> DISMISS["Student reads support content
Dismisses popup or taps next word"]
    DISMISS --> CONT([Returns to transcript — repeat as needed]):::endnode

    EVENT --> PROFILE["Difficulty profile builds over time
Tracks repeated lookups across sessions"]
    PROFILE --> VOCAB["Personalized vocabulary lists
generated from difficulty history"]
    VOCAB --> DASH["Feeds into Data Dashboard
(Flow 4, Section B)"]:::info
    DASH --> DONE([Teachers see student difficulty data]):::endnode

C Difficulty Events → Data Dashboard

100%
flowchart TD
    START([Student interacts with transcript]):::start --> EVENTS["Difficulty events collected:
Word clicks from ClickableTranscript
+ Mirroring practice errors
+ Content mirroring retries"]

    EVENTS --> STORE["Events stored per student
word · timestamp · source
sessionId · contentArea"]

    STORE --> AGGREGATE["Backend aggregation engine
Processes events periodically"]:::ai

    AGGREGATE --> IND_METRICS["Per-Student Metrics:
Language Access Level %
Difficulty Rate %
Completion Rate
Retry Intensity"]
    AGGREGATE --> GROUP_METRICS["Group Metrics:
Class-wide difficulty words
Average access level by grade
Distribution by proficiency tier"]

    IND_METRICS --> DASH["Data Dashboard
(Flow 4 — Section B)"]
    GROUP_METRICS --> DASH

    DASH --> SA{Viewer role?}:::decision
    SA -->|School Admin| SA_VIEW["Full dashboard access
All students, all classes
Filter by grade · subject · WIDA/ELPAC"]
    SA -->|ESL Teacher| ET_VIEW["Dashboard for assigned classes
Individual + group data
RTI tier recommendations"]

    SA_VIEW --> ACTION["Informed decisions:
Which students need Tier 2/3 support
Which content areas are hardest
Progress trends over time"]
    ET_VIEW --> ACTION

    ACTION --> DONE([Data-driven instruction]):::endnode
